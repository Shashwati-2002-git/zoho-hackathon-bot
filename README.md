Zoho Hackathon Bot is basically a Zobot that embeds into any website, scrapes out all the information given on it and guides
the website visitors through the website, interacts with them and answers to their questions for a better user experience.

File structure:
/zoho-hackathon-bot
   |-- node_modules/
   |-- public/
   |     |-- index.html
   |     |-- widget.js
   |-- package.json
   |-- server.js
